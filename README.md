Password Strength Checker
This tool helps you test password strength in real time. You type a password, and the app scores it with a clear meter. You see length, character mix, repeated patterns, and other risks. You get simple tips to improve your security.

Features
Real-time strength meter
Checks length, upper and lower letters, numbers, and symbols
Flags repeated or sequential characters
Warns about common weak passwords
Gives clear steps to improve
Works offline in any browser

How to use
Open the page in your browser. Type any password into the field. Read the score and follow the tips.

Tech
Pure HTML, CSS, and JavaScript. No backend. No data stored.
